## Bienvenido a CPA Virtual

Este manual le guiará paso a paso en el uso de la aplicación CPA Virtual, la plataforma digital de su Centro de Participación Activa (CPA).

### ¿Qué es CPA Virtual?

CPA Virtual es la aplicación que le permite acceder de forma sencilla y desde cualquier lugar a todos los servicios de su centro:
- Consultar y apuntarse a talleres y actividades
- Solicitar citas para servicios
- Gestionar sus inscripciones
- Ver su agenda personal
- Acceder a sus datos y documentación

## 📋 Convenciones del Manual

### Elementos de la interfaz

- **Botones**: Se muestran entre comillas → Hacer clic en "Aceptar"
- **Campos de texto**: Se indican en cursiva → Introducir su *nombre de usuario*
- **Menús**: Se separan con el símbolo → → Menú → Mi Área → Mis Datos
- **Enlaces**: Se muestran subrayados → Más información

### Formato de instrucciones

Las instrucciones se presentan en listas numeradas:

1. Primer paso a realizar
2. Segundo paso a realizar
3. Tercer paso a realizar

Las opciones disponibles se presentan con viñetas:

- Opción A
- Opción B
- Opción C
